\chapter{滤波、调制和采样}

我们要求信号频谱 $[f_L, f_H]$ 必须位于第 $m$ 个奈奎斯特区间内。这个区间的范围是：$$\text{下界} = (m-1)\frac{f_s}{2} \quad \text{与} \quad \text{上界} = m\frac{f_s}{2}$$为了让信号 $[f_L, f_H]$ 也就是整个“车”停进这个“车位”里，必须同时满足两个条件：条件 1：信号没超出上界（车头没越线）$$f_H \le m\frac{f_s}{2}$$移项解出 $f_s$：$$2f_H \le m f_s \implies \boldsymbol{f_s \ge \frac{2f_H}{m}}$$这就是公式的下限。条件 2：信号没超出下界（车尾没越线）$$f_L \ge (m-1)\frac{f_s}{2}$$移项解出 $f_s$：$$2f_L \ge (m-1) f_s \implies \boldsymbol{f_s \le \frac{2f_L}{m-1}}$$这就是公式的上限。合并两个条件：$$\frac{2f_H}{m} \le f_s \le \frac{2f_L}{m-1}$$补充约束：关于 $m$ 的取值为了保证上述不等式有解（即上限必须大于等于下限），我们需要：$$\frac{2f_H}{m} \le \frac{2f_L}{m-1}$$化简这个不等式（代入 $f_H = f_L + B$）：$$\frac{f_L + B}{m} \le \frac{f_L}{m-1}$$$$(f_L + B)(m-1) \le f_L m$$$$f_L m - f_L + Bm - B \le f_L m$$$$Bm \le f_L + B = f_H$$$$m \le \frac{f_H}{B}$$因为 $m$ 必须是整数，所以 $m$ 的最大取值为：$$m_{max} = \left\lfloor \frac{f_H}{B} \right\rfloor$$($\lfloor \cdot \rfloor$ 表示向下取整)
�� 核心概念解析
对于一个频率范围为 $[f_L, f_H]$，带宽为 $B = f_H - f_L$ 的带通信号，为了避免混叠，采样频率 $f_s$ 需满足以下公式：$$\frac{2f_H}{m} \le f_s \le \frac{2f_L}{m-1}$$其中，$m$ 是一个正整数，取值范围为：$$1 \le m \le \left\lfloor \frac{f_H}{B} \right\rfloor$$注：$\lfloor \cdot \rfloor$ 表示向下取整。�� 详细计算步骤：确定参数：找出信号的最高频率 $f_H$ 和最低频率 $f_L$，计算带宽 $B = f_H - f_L$。计算 m 的最大值：$$m_{max} = \left\lfloor \frac{f_H}{B} \right\rfloor$$这意味着你可以把频谱“折叠”多少次而不发生重叠。寻找可行的采样区间：对于 $m = 1, 2, ..., m_{max}$，代入不等式算出 $f_s$ 的允许范围：当 $m=1$ 时（即基带采样）：$2f_H \le f_s < \infty$当 $m=m_{max}$ 时：得到最低的允许采样频率范围。确定最小采样频率：通常题目问的“最小采样频率”位于 $m = m_{max}$ 对应的区间的下限，即：$$f_{s,min} = \frac{2f_H}{m_{max}}$$注意：理论上的极限最小值是 $2B$，但只有当 $f_H$ 是 $B$ 的整数倍时，才能精确取到 $2B$。大多数情况下，$f_{s,min}$ 会略大于 $2B$。
